FETCH

File Extension Type CHecker

FETCH is a tool designed to verify the authenticity of files and their extensions by analyzing their's magic numbers (binary signatures). This helps identify potentially malicious files that have been disguised with misleading extensions.

How it works:

First, upload a file using the form below. FETCH will examine the file extension and read the file's magic number before comparing the two and reporting any mismatches.
